Compact API Cheat-Sheet
Users: GET /api/users/, POST /api/users/, DELETE /api/users/{userId}
Categories: GET /api/categories/, POST /api/categories/, DELETE /api/categories/{catId}
Posts: GET /api/posts, GET /api/posts/{postId}, POST /api/user/{userId}/category/{categoryId}/posts, DELETE /api/posts/{postId}
Comments: POST /api/post/{postId}/comments, DELETE /api/comments/{commentId}
